# Rehabilitation-software-
 A development software for Joint Rehabilitation Device (JRD).
 Software will connect JRD throught USB cable or board RF CC1101.

## Enviroment Develop
 Qt 5.6.1 or later.
 Qt Widget module/C++ language.
